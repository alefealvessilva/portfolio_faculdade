Verify: Anti-Fraude Pix - Android Mobile APP

Desenvolvi o aplicativo Verify: Anti-Fraude Pix que oferece uma solução prática e segura para a gestão de transações Pix, especialmente voltada para pessoas jurídicas. O app integra-se de forma simples com a API Pix, permitindo que os usuários confirmem recebimentos e evitem fraudes.

Funcionalidades Principais

  • Autenticação Simplificada: Suporte para login com Google, e-mail e senha, garantindo segurança e facilidade de uso.
  • Cadastro de Múltiplos Bancos: Permite o cadastro de contas de diferentes bancos, como Banco Sicoob e Banco do Brasil, com a possibilidade de reposicionar o banco preferido na tela principal.
  • Visualização de Transações: Exibe transações recentes na página inicial e oferece uma timeline para consulta de todas as transações.
  • Sincronização em Nuvem: Dados sincronizados em nuvem, permitindo o uso do app em múltiplos dispositivos sem necessidade de recadastro.
  • Interface Intuitiva: Desenvolvida com componentes do Material Design 3, proporcionando uma experiência de usuário moderna e intuitiva.

Tecnologias Utilizadas

  • Flutter/Dart: Para o desenvolvimento de uma interface de usuário responsiva e eficiente.
  • APIs Pix: Integração com APIs do Banco do Brasil e Banco Sicoob para gestão de transações.
  • Firebase: Sincronização segura dos dados dos usuários.
  • Sincronização em Nuvem: Dados sincronizados em nuvem, permitindo o uso do app em múltiplos dispositivos sem necessidade de recadastro.

O Verify é uma ferramenta essencial para empresas que precisam gerenciar transações Pix de forma rápida e segura, garantindo tranquilidade no recebimento de pagamentos e oferecendo a melhor experiência para os clientes.
